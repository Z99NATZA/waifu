transport -> usecase -> domain <- infra

transport/handler ส่ง infra เข้า usecase
            
------------------------------------------------------
            
# Domain = กฎหมาย
- ไม่วิ่งไปหาศาล
- ไม่จับคน
- แค่บอกว่าอะไรผิด / ถูก

# Application = ผู้พิพากษา
- อ่านกฎหมาย
- ตัดสินใจ flow
- สั่งให้ infra ทำงาน

# Infrastructure = ตำรวจ
- ทำตามคำสั่ง
- ไม่มี logic ธุรกิจ

# Transport = ประตูศาล
- รับคำร้อง
- แปลงรูปแบบ
    
------------------------------------------------------
